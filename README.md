# Summary 
This solution includes:

##  HashValues
 A console App which allows a user to hash all files in a directory.  Takes the 'Source Path' and 'Destination Path' as user inputs. 

## HashFilesLib
 A class library including methods used to hash files. 

## HashFilesApp
 A WPF App which allows a user to hash a specific file or all files in a directory. Takes 'Source Path' and 'Source file name' as user inputs. Hashed files will be found within the 'Source Path' directory in a fold called 'Output'. 
 
### Inputs
Files to be parsed can be found here: 
- C:\Training\CSharpTraining\Hawthorn Xerox Sample OPO files\Input
  
## X# console application 
 A console application that provides an example of object oriented programming in XSharp.  Includes Vehicle Class which parents Car, Motorbike and Bus. The application creates instances of each object and adds them to arrays and lists.  Then completes different operations to show how the data can be accessed.
